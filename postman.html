Posts API

Start the backend first (cd backend && npm run dev). Base URL:

http://localhost:3001

  • GET /posts — list all
  • GET /posts/:id — one post
  • POST /posts — JSON body: title, body, userId
  • PUT /posts/:id — JSON body: title, body, userId
  • PATCH /posts/:id — JSON body: any subset of fields
  • DELETE /posts/:id
